# ProxyPilot

Мобильное приложение для Android.

## О приложении

ProxyPilot - это современное приложение с удобным интерфейсом и широкими возможностями настройки.

## Основные функции

- Простой и интуитивный интерфейс
- Поддержка различных конфигураций
- Автоматическое обновление настроек
- Работа в фоновом режиме
- Минимальное потребление батареи

## Системные требования

- Android 5.0 и выше
- 50 МБ свободного места
- Доступ к интернету

## Установка

1. Скачайте APK файл
2. Разрешите установку из неизвестных источников
3. Установите приложение
4. Запустите и следуйте инструкциям

## Обновления

Приложение автоматически проверяет наличие обновлений при запуске.
